HA-TFF: Hardware-Accelerated Telemetry Firewall

[README] [ARCHITECTURE.MD]
The Engineering Bottleneck

Line-rate 100GbE processing requires evaluating minimum-size (64-byte) frames every 6.72 nanoseconds. Software stacks fail entirely at this inter-packet arrival constraint.

The Architecture

A pure-RTL, 16-cycle exact-match datapath firewall targeting FPGAs. Utilizes a 512-bit AXI4-Stream bus for combinatorial header extraction, eliminating the need for complex state machines or byte-by-byte buffering.

Structural Trade-offs

I implemented 4-way parallel BRAM lookups using Cuckoo Hashing to guarantee O(1) exact-match lookup time, deliberately accepting a 48-block BRAM utilization cost over a slower linked-list collision resolution mechanism. I also maintained a strict zero-DSP footprint to maximize cross-platform portability across Xilinx families.

EDP: Embedded Design Platform Middleware

[README] [ARCHITECTURE.MD]
The Engineering Bottleneck

Tightly-coupled data planes in modern cyber-physical systems (like drones) exhibit O(N²) complexity as new protocols (MAVLink, ROS2) are added. Memory copying across these monolithic bounds induces critical latency spikes.

The Architecture

A deterministic, zero-copy IPC C11 middleware designed to separate protocol logic from transport layers. It utilizes shared memory and pointer passing via highly synchronized ring buffers to achieve zero-allocation hot paths.

Structural Trade-offs

To prevent context-switching overhead in the critical path, I implemented lock-free atomics rather than standard POSIX mutex synchronization. This increased implementation complexity but guaranteed deterministically bounded IPC latency across embedded targets.

SiliconForge: Analog-to-UVM EDA Engine

[README] [ARCHITECTURE.MD]
The Engineering Bottleneck

Transient analog SPICE simulation is exceptionally slow, prohibiting exhaustive digital regressions. However, standard heuristic noise injection in digital simulations is mathematically unsound for non-linear distortion (e.g., PLL phase noise).

The Architecture

An experimental Python-based EDA tool utilizing Jinja2 AST templating to extract SPICE numerical matrices (Perturbation Projection Vectors) and translate them into deterministic SystemVerilog/UVM constraint models and coverage groups.

Structural Trade-offs

Replaced continuous-time dynamic simulation with static functional coverage metrics bound by Periodic Steady State (PSS) waveforms. This trades raw physical resolution for mathematical determinism and integration into high-velocity CI/CD pipelines.
